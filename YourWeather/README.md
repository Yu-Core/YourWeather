# 项目结构

- YourWeather.Maui `Maui Blazor`
- YourWeather.WebAssembly `Blazor WebAssembly`
- YourWeather.Photino `Photino Blazor`
- YourWeather.Rcl `Razor类库，存放页面和一些基础实现`
- YourWeather.Rcl.Desktop `Winform和WPF共用的一些实现类`
- YourWeather.Rcl.Web `WebAssembly 和 Server 共用的一些实现类`
- YourWeather.Server `Blazor Server`
- YourWeather.Shared `共用的实体`
- YourWeather.Winform `Winform Blazor`
- YourWeather.Wpf `WPF Blazor`
