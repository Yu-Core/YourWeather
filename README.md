# YourWeather

「YourWeather」是一个支持多平台的天气应用，可以在Web端、桌面端、移动端运行。

<img src="./Images/Weather.png" />

## 信息
- 语言：C#
- 主要技术：Blazor Wasm、MAUI Blazor
- .NET版本：.NET 6

## 支持的平台
- Web
- Windows
- Android
- ~~iOS/Mac~~ (由于没有苹果设备可以测试，能否运行未知)
